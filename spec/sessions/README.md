# Sessions

Start here for short-lived continuity between agent sessions.

- `HANDOFF.md` — compact continuation note for the next agent
- `session-XX.md` — numbered session logs; read the highest-numbered open file first
- `templates/SESSION_TEMPLATE.md` — template for new session logs
- `templates/HANDOFF_TEMPLATE.md` — template for handoff notes

Use this folder for verified session state only.
Do not turn it into a transcript archive or a second project spec.
